## Installation

1. Clone the repository:

```bash
git clone https://github.com/Shazzadhossensunny/ezymetrics-dashboard.git

```

2. Install dependencies:

```bash
npm install
```

## Running the Project

```bash
npm run dev
```

The application will be available at `http://localhost:5173`
